# Quickstart: サイドバーとビューアのスクロール動作修正

## 前提
- ブランチ: `003-fix-sidebar-scroll`
- 仕様: [spec.md](./spec.md)
- 計画: [plan.md](./plan.md)

## 手順（実装）
1. **レイアウト基盤**: `App.tsx` および関連レイアウトコンポーネントでヘッダー直下のコンテナを `flex` にし、子（サイドバー/ビューア）に `min-h-0` を適用。高さは `h-[calc(100vh-<header>)]` などでCSS計算。
2. **サイドバー**: サイドバーラッパーに `overflow-y-auto` と `min-h-0` を付与し、ビューアとは独立したスクロール領域にする。
3. **ビューア**: ビューアラッパーをデフォルト `overflow-hidden` にし、ズーム倍率が等倍を超えた場合のみ `overflow-auto` を適用。画像切替時にスクロール位置をリセット。
4. **リサイズ対応**: 追加のJSリスナーを置かず、CSSレイアウトでリサイズに追従するか確認。
5. **テスト**: jsdom + Testing Libraryで `overflow` の切替、スクロールリセット、サイドバー独立スクロールを検証。

## 手順（品質確認）
- `pnpm type-check`
- `pnpm lint`
- `pnpm test`
- （必要に応じて）`pnpm build`

## 期待動作チェック
- 通常表示でビューアにスクロールバーが出ない
- サイドバーのみスクロールし、ビューアは固定
- ズームイン時のみビューアがスクロール可能
- ウィンドウリサイズ後も高さ同期が維持
